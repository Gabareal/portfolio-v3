My Projects

Lost & Found site

See it here!

This was a team project done in Sec 3 with 3 of my friends. The task was to design a digital solution to solve a real-world problem, so our idea was to come up with an accessible lost-and-found app! Although it was never implemented to an actual solution, it taught me a lot about javascript and firebase! It was one of my first times using an API and interacting with data storage, login systems with cookies, etc.
As you can see, the UI is very "minimalist" at best. Some buttons may not work any more as the firebase token has expired, but feel free to look through the code! (if you want to)

Camera-based file selector DEMO

See it here!

I was inspired by how Iron Man uses his hands in the air to select stuff on his holographic table, so this site is an attempt of recreating that same effect, but at its very barebones. Although it is one of my cooler mini-projects, a lot of code was made with the help of generative AI, with the UI being entirely AI generated. So take that as you will.
